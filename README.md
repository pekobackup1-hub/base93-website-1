# Base93 Landing Page

Modern Next.js landing page for Base93, built with TypeScript, App Router, and Tailwind CSS.

## Local Commands

```bash
npm install
npm run dev
npm run build
npm run start
```

## Brand Notes

- Primary color: Deep Purple `#2B2441`
- Supporting colors: Light Purple `#DDD2FF`, Dark Lavender `#514577`, Amethyst `#7D68C0`, Light Grey `#ECEFF2`
- Dark logo is used on light backgrounds.
- White logo is used on dark backgrounds.
- The logo is not recolored, distorted, rotated, or given gradients.
